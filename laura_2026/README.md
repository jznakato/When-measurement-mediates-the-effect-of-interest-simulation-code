# When-measurement-mediates-the-causal-effect-of-interest-simulation-code

This repository contains the code for implementation of the estimators in "A causal framework for strategies aiming to improve health screening and health outcomes" by Nakato et al. 

The corresponding manuscript is available at https://arxiv.org/abs/2506.06267.
